Week One:FOUNDATIONS

Welcome to the foundational week of my Web Foundation Journey! During Week 1, I delved into the core elements of web development, gaining insights into the fundamental technologies that power the internet.

Learning Outcomes

By the end of Week 1, I’ve acquired the following skills:
Understanding the Basics:

Explored the fundamentals of web development, including the roles of HTML, CSS, and JavaScript.

Gained insights into how these technologies work together to create dynamic and interactive webpages.

HTML Mastery:

Learned the essentials of HTML, including elements and attributes. Applied this knowledge to structure a webpage effectively.

CSS Styling:

Explored CSS selectors and properties to style HTML elements. Implemented styles to enhance the visual appeal of web content.

Overview of my week one Foundation

Introduction to JavaScript:

Received a taste of JavaScript, understanding its role in making webpages dynamic. Used the script tag to include a JavaScript file on a webpage.

Browser Devtools:

Explored the powerful tools provided by web browsers for debugging and development.

To witness my progress and explore the practical applications of Week 1 concepts, feel free to check out the following:

HTML Elements Practice Exercise - Test your understanding of HTML elements.

CSS Styling Project - See how I applied CSS styling to enhance webpage aesthetics.

Course Outline

This week's journey covered the following topics:
  • Intro to Web Development, the internet, and the web.
  • HTML Elements and Attributes.
  • CSS Selectors and Properties.
  • A Taste of JavaScript.
  • Browser Devtools.
  • Completed Tasks and Project

Embark on this journey with me, and let's continue building a strong foundation for the exciting weeks ahead!
